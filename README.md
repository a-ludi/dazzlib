Install
--------

Install using [`dub`](https://code.dlang.org/)

```sh
dub add dazzlib
```

or by manually adding the dependency:

```
# dub.json:
"dazzlib": ">=0.0.1"

# dub.sdl:
dependency "dazzlib" version=">=0.0.1"
```
